CSS Quick Tips

Master Visualization in CSS

1. Create a design file. Have a CSS file that is not linked to your HTML file where you store your most used colors, fonts, etc. for easy access and better organization.

2. Create a universal CSS file. Universal CSS files may be helpful when you have a larger site spread across multiple HTML pages. HTML Do's and Dont's

Take your HTML from beginner to intermediate with these tips

1. Start with the basics. You should start your HTML file using the standard H1 through H6 headings and other predesignated HTML structural elements. This will help reduce overcrowding of classes and IDs. This also aids in web browser crawling, which is when the web browsers filter through your HTML sheet and look for relevant information to present upon query searches.

2. Create classes that are widely useable. The idea of classes is that they apply to more than one instance of an element. By creating classes that are widely used you will inevitably create a more cohesive style amongst your site.

Understanding Logic (The easy way)

You use logic everyday, let's make it simpler

There are four main logic operators.

  • AND &
  • XOR ^
  • OR |
  • NOT ~

The symbols used for these operators may vary in text, but in coding practice these four symbols are what you will use to code logic.

Let's think about logic in the form of a search engine.

AND - Two truths make a truth.

ex: If your search asks for something to be blue AND round the only results which would return true must be both blue AND round. If an item is blue but NOT round, this would be false.

In binary terms: if we were to do 1100 & (AND) 0101 our result would be 0100.

XOR - A truth and a lie makes a truth.

ex: If your search asks for something to be blue XOR round the only true results which would return true must be either blue OR round. Search results that are both blue AND round would return false for XOR.

In binary terms: if we were to do 1100 ^ 0101 our result would be 1001.

OR - A truth and a lie or a truth and a truth makes a truth.

ex: If your search asks for someting to be blue OR round the results which would return true would be items which are blue and round, or just blue, or just round.

In binary terms: if we were to do 1100 | 0101 our result would be 1101.

NOT - Only what is not true makes a truth.

ex: If you search asks for something to be NOT blue the only results which would return true would be items which are NOT blue.

In binary terms: if we were to do ~1100 our result would be 0011.
